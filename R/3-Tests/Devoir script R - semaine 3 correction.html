1. Le pourcentage de sujets recommandant le service dans lequel ils sont passés est-il significativement différent chez les hommes et chez les femmes ?

Nous sommes dans un tableau 2 X 2 donc nous allons utiliser le test du chi2

Recodage des variables catégorielles effectué en semaine 1 et 2

satisf$sexe.c <- factor(satisf$sexe, labels = c("homme", "femme"))
satisf$recommander.b <- ifelse(satisf$recommander > 1, 1, 0)

Tableau croisé entre recommander.b et sexe (en %)

tab <- table(satisf$recommander.b, satisf$sexe.c, deparse.level = TRUE, useNA = "always")
prop.table(tab)

##       
##         homme  femme   <NA>
##   0    0.1367 0.1180 0.0000
##   1    0.2603 0.2434 0.0000
##   <NA> 0.1049 0.1367 0.0000

Condition de validité

Pour appliquer le test du chi-deux, il faut que les effectifs espérés soit supérieurs à 5. Sinon un test de Fisher sera plus approprié.

chi2 <- chisq.test(satisf$recommander.b, satisf$sexe.c, correct = FALSE)
chi2$expected

##                     satisf$sexe.c
## satisf$recommander.b  homme  femme
##                    0  71.19  64.81
##                    1 140.81 128.19

Les effectifs espérés sont tous supérieurs à 5 donc le test du chi2 est applicable.

Application du test statistique

chi2

## 
##  Pearson's Chi-squared test
## 
## data:  satisf$recommander.b and satisf$sexe.c
## X-squared = 0.1454, df = 1, p-value = 0.703

Le degré de significativité (p-value) associé au test du chi-deux est très élevé (p = 0,703), il n'y a donc pas de raison de penser que les femmes et les hommes recommandent le service de manière différente. En d'autres termes, la différence observée entre le pourcentage de femmes recommandant le service et le pourcentage d'hommes recommandant le service est due au hasard.

2. La moyenne du score de relation est-il significativement différent chez les hommes et chez les femmes ?

Nous travaillons avec une variable catégorielle et une variable continue donc nous allons utiliser le test de Student.

Condition de validité

Pour appliquer un test de student, on doit avoir au moins 30 personnes par groupe et la variable à expliquer doit suivre une loi normale. Sinon un test de Wilcoxon sera plus approprié.

table(satisf$sexe, useNA = "always")

## 
##    0    1 <NA> 
##  268  266    0

Les effectifs de chaque groupe sont supérieurs à 30. Cette condition de validité est donc vérifiée.

hist(satisf$score.relation, main = "Histogramme de la variable score.relation", 
    xlab = "score.relation", ylab = "Nombre de sujets", col = rgb(223, 241, 
        203, maxColorValue = 255))


qqnorm(satisf$score.relation, main = "Diagramme QQ pour le score de relation/ loi normale", 
    xlab = "quantiles théoriques", ylab = "quantiles empiriques")
qqline(satisf$score.relation)

L'hypothèse de normalité n'est pas vérifiée, la distribution du score est très asymétrique. On préfère donc utiliser un test de Wilcoxon.

Application du test statistique

wilcox.test(satisf$score.relation ~ satisf$sexe)

## 
##  Wilcoxon rank sum test with continuity correction
## 
## data:  satisf$score.relation by satisf$sexe
## W = 16072, p-value = 0.3454
## alternative hypothesis: true location shift is not equal to 0

La p-value vaut 0,345. La p-value est assez élevée, la différence de moyenne du score de relation entre les hommes et les femmes n'est donc pas significative.

3. Le score de relation est-il significativement corrélé à l’âge ?

Condition de validité

Pour appliquer un test de corrélation, il faut qu'au moins une des deux variables suive une distribution normale pour utiliser la fonction cor.test.

hist(satisf$age, main = "Histogramme de la variable Age", xlab = "Age", ylab = "Nombre de sujets", 
    col = rgb(223, 241, 203, maxColorValue = 255))


qqnorm(satisf$age, main = "Diagramme QQ pour l'âge / loi normale", xlab = "quantiles théoriques", 
    ylab = "quantiles empiriques")
qqline(satisf$age)

La distribution de l'âge peut être modélisée par une loi normale.

Application du test statistique

cor.test(satisf$score.relation, satisf$age)

## 
##  Pearson's product-moment correlation
## 
## data:  satisf$score.relation and satisf$age
## t = 1.796, df = 347, p-value = 0.07336
## alternative hypothesis: true correlation is not equal to 0
## 95 percent confidence interval:
##  -0.009102  0.198945
## sample estimates:
##     cor 
## 0.09597

La p-value vaut 0,073, ce qui est relativement élevé (p-value > 0,05). On ne peut donc pas considérer que le score de relation est significativement corrélé à l'âge.
