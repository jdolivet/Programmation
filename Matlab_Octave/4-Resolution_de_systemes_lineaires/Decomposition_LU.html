Soient a⃗  et b⃗  deux N-vecteurs donnés, c⃗  et d⃗  deux (N−1)-vecteurs donnés. On considère la N×N-matrice A définie par A=⎛⎝⎜⎜⎜⎜⎜⎜⎜⎜⎜a1d1c1a2d2c2a3⋱c3⋱⋱⋱⋱dN−1cN−1aN⎞⎠⎟⎟⎟⎟⎟⎟⎟⎟⎟
et on veut résoudre le système linéaire Ax⃗ =b⃗  en utilisant la décomposition LU de A. Pour ce faire, on propose l'algorithme (octave, matlab) suivant que l'on complètera. Vous pouvez sauver le fichier declu.m, le compléter et le tester avec octave, puis répondre au questions ci-dessous.
function [err,b]=declu(N)
%
% Algorithme de decomposition A=LU
% pour resoudre le systeme lineaire Ax=b,
% ou A est la matrice tridiagonale
% de diagonale a, sur-diagonale c et sous-diagonale d.
%
% parametres
%
% N : nombre d inconnues du systeme lineaire
% a : N-vecteur, diagonale de A puis diagonale de L
% c : (N-1)-vecteur, sur-diagonale de A en entree, puis sur-diagonale de U
% d : (N-1)-vecteur, sous-diagonale de A, puis sous-diagonale de L
% b : N-vecteur, second membre du systeme lineaire en entree, solution du systeme lineaire Ax=b en sortie
%
% remplissage de la matrice A et du second membre b
%
for i=1:N
 a(i)=3.;
 b(i)=0.;
end
b(1)=2. ;
b(N) = 1.;
for i=1:(N-1)
 c(i)=-1.;
 d(i)=-2.;
end
%
% algorithme de decomposition A=LU
%
c(1) = c(1)/a(1);
for i=2:(N-1)
 a(i)=a(i)-c(i-1)*???;
 c(i)=c(i)/???;
end
a(N)=a(N)-???;
%
%
%
% resolution du systeme lineaire Ly = b
%
b(1)=b(1)/a(1);
for i=1:N-1
 b(i+1)=(b(i+1)-d(i)*???)/a(i+1) ;
end
%
% resolution du systeme lineaire U x = y
%
for i=N-1:-1:1
 b(i)=b(i)-c(i)*???;
end
%
% calcul de l'erreur
%
err=0.;
for i=1:N
 err=err+(b(i)-1)^2;
end
err=sqrt(err);
end
